nene
====

Remove ANSI escape codes in file.

output to stdout:

```shell
$ nene xxx.log
...
```

output to file:

```shell
$ nene xxx.log --out new_xxx.log
output: new_xxx.log
```

## Install from Source

```shell
$ git clone https://github.com/YusukeHosonuma/nene.git
$ cd nene
$ cargo install
```
